# 慢性阻塞性肺疾病（COPD）诊疗指南

## 1. 疾病定义与诊断标准

慢性阻塞性肺疾病（Chronic Obstructive Pulmonary Disease, COPD）是一种以持续气流受限为特征的可预防和可治疗的疾病。诊断依据肺功能检查：使用支气管舒张剂后，FEV1/FVC < 0.70即可确诊。

### 1.1 GOLD分级标准

根据FEV1占预计值百分比分为4级：
- GOLD 1级（轻度）：FEV1 ≥ 80%预计值
- GOLD 2级（中度）：50% ≤ FEV1 < 80%预计值
- GOLD 3级（重度）：30% ≤ FEV1 < 50%预计值
- GOLD 4级（极重度）：FEV1 < 30%预计值

### 1.2 症状评估

mMRC呼吸困难量表评分：
- 0分：仅在剧烈活动时出现呼吸困难
- 1分：平地快走或爬缓坡时出现呼吸困难
- 2分：因呼吸困难而比同龄人走得慢，或需停下来休息
- 3分：平地行走100米或数分钟后需停下来喘气
- 4分：因严重呼吸困难而不能离开房间，或穿脱衣服时出现呼吸困难

CAT评分（COPD评估测试）：总分0-40分，≥10分提示症状负担较重。

## 2. 药物治疗方案

### 2.1 支气管舒张剂

**短效β2受体激动剂（SABA）**：
- 沙丁胺醇（Salbutamol）：100-200 μg/次，按需使用，每日不超过8次
- 起效时间：5-15分钟
- 持续时间：4-6小时

**长效β2受体激动剂（LABA）**：
- 福莫特罗（Formoterol）：12 μg，每日2次
- 茚达特罗（Indacaterol）：150 μg，每日1次
- 持续时间：12-24小时

**长效抗胆碱能药物（LAMA）**：
- 噻托溴铵（Tiotropium）：18 μg，每日1次
- 格隆溴铵（Glycopyrronium）：50 μg，每日1次
- 起效时间：30-60分钟
- 持续时间：24小时
