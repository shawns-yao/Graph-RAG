if not norm in merged:

## Task 5: reflection overall_score 聚合重构 — 从 min 改为加权平均

**Files:**
- Modify: `packages/rag-core/rag_core/reflector.py:536-544`
- Test: `packages/rag-core/tests/test_reflector.py`

- [ ] **Step 1: 修改 overall_score 计算方式**

将 `min(relevance, entity_completeness, logical_consistency, context_sufficiency)` 改为加权平均：

```python
overall_score = round(
    0.35 * relevance
    + 0.30 * entity_completeness
    + 0.15 * logical_consistency
    + 0.20 * context_sufficiency,
    3,
)
```

理由：entity_completeness 和 relevance 最关键，logical_consistency 最不重要（evidence 内部一致性通常较高）。

- [ ] **Step 2: 新增 reflection-to-confidence 映射函数**

在 `reflector.py` 新增函数：

```python
def reflection_to_confidence(reflection: ReflectionStep) -> float:
    """Map reflection 0-5 scores to 0-1 confidence.
    
    Uses the same weighted average as overall_score, then maps to [0.1, 1.0].
    """
    if reflection.overall_score <= 0:
        return 0.0
    weights = {"relevance": 0.35, "entity_completeness": 0.30,
               "logical_consistency": 0.15, "context_sufficiency": 0.20}
    raw_score = (
        weights["relevance"] * reflection.relevance
        + weights["entity_completeness"] * reflection.entity_completeness
        + weights["logical_consistency"] * reflection.logical_consistency
        + weights["context_sufficiency"] * reflection.context_sufficiency
    )
    # Map 0-5 to 0.1-1.0 (score 4.0+ maps to 0.9+)
    confidence = min(1.0, max(0.1, raw_score / 5.0))
    return round(confidence, 3)
```

# Fusion: 50% retrieval quality + 50% reflection quality

    fused_score = 0.5 * retrieval_score_normalized + 0.5 * reflection_score
    confidence = min(1.0, max(0.1, fused_score / 5.0))
